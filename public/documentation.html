Movie Database API

A server-side component of a 'movies' web application. This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Logic: URL: Method: Request body data format: Response body data format:
Return a list of all movies /movies GET None None
Return data about a movie /movies/:movieTitle GET None A JSON object holding data about the requested movie.
Return data about a genre /movies/genre/:genreName GET None A JSON object holding data about the requested genre.
Return data about a director /movies/directors/:directorName GET None A JSON object holding data about the requested director.
Allow new users to register /users POST A JSON object holding data about the user to add. A JSON object holding data about the user that was added.
Allow users to update their user info /users/:id PUT A JSON object holding new user data. A JSON object holding of the updated user data.
Allow users to add to their favorite movies /users/:id/:movieTitle POST None A message indicating the movie was removed.
Allow users to delete from their favorite movies /users/:id/:movieTitle DELETE None A message indicating the movie was removed.
Allow users to deregister /users/:id DELETE None A message indicating the user was deregistered.
